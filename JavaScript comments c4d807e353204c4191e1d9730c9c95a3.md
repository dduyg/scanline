# JavaScript comments

- *Single-Line Comment with Separator:*

```jsx
// =========================
//      Topic Introduction
// =========================
```

- *Multi-Line Comment with Separator:*

```jsx
/*******************************
          Topic Introduction
*******************************/
```

- *Banner Comment with ASCII Art:*

```jsx
/**********************************************
 *                                            *
 *              Topic Introduction            *
 *                                            *
 **********************************************/
```
